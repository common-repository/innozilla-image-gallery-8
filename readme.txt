== Description ==

Innozilla Image Gallery 8 - is the simplest plugin to integrate WordPress builtin Photo Galleries into your pages with excellent visual effects.

**How to use:**

Using **[isotope_gallery]** shortcode, you can publish isotope image gallery into any Page or Post on your WordPress sites.

[youtube https://www.youtube.com/watch?v=8A-2mYCL-Rk ]

**Important:**

[DEMO VERSION](https://innozilla.com/wordpress-plugins/image-gallery-8/ "Your favorite gallery")

**Features**

* Filter Functionality
* Load more Effect
* Fully Responsive
* Lightbox Image Preview

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/innozilla-image-gallery-8.zip` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin.
3. How to use? please check this video [HOW TO USE THE PLUGIN](https://www.youtube.com/watch?v=8A-2mYCL-Rk "Demo video")

== Frequently Asked Questions ==

= How to add Gallery to website post or page? =

Simply paste this [isotope_gallery] shortcode to any post or page on your WordPress site.

= Is the Gallery responsive? =

Yes, the Gallery is absolutely responsive. No matter what device you will use.
